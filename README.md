## Restaurant Table Reservation – DOM Practice

This project is a small exercise built using vanilla JavaScript and the DOM only.

### What it does
- Displays available restaurant tables as buttons
- Tables are created dynamically from JavaScript data
- Only one table can be selected at a time
- The selected table changes color
- Reservation details are shown when a table is selected

### What I practiced
- Creating elements with the DOM
- Using event listeners
- Handling basic state
- Updating the UI based on user interaction

### Tech used
- JavaScript
- DOM API
- Basic CSS
